# SAP CO Module

## Overview

This folder contains the Controlling (CO) component of the SAP S/4HANA-inspired ERP implementation.

The CO module focuses on internal cost management, budget control, profitability analysis, and management reporting to support operational and financial decision-making within hospitality operations.

## Scope

The implementation replicates selected SAP CO business processes using Python, SQLite, and Power BI.

Key areas include:

* Cost Center Accounting
* Budget vs Actual Analysis
* Profitability Analysis
* Management Reporting

## SAP References

The project is inspired by the following SAP CO functionalities:

* CO-CCA (Cost Center Accounting)
* CO-PA (Profitability Analysis)
* Cost Monitoring and Reporting
* Budget Control and Variance Analysis

## Key Features

### Cost Center Accounting

* Cost center structure design
* Cost allocation and monitoring
* Cost trend analysis

### Budget Control

* Budget planning and tracking
* Budget vs actual comparison
* Variance calculation and reporting

### Profitability Analysis

* Profitability by service category
