# Project Scope and Objectives

## Scope

Create a SAP S/4HANA-inspired hospitality ERP analytics prototype that integrates FI, CO, SD, MM, analytics, BI, and documentation deliverables using synthetic/anonymized sample data.

## Objectives

- Build SQL-ready CSV data for core hospitality ERP processes.
- Automate FI, CO, SD, MM, and forecasting analysis with Python.
- Produce text-based reports, KPI tables, SVG charts, and an HTML dashboard.
- Keep the repository suitable for GitHub, interviews, and portfolio review.
- Avoid confidential data and avoid claims of a real SAP implementation.
