# Data Preparation

## Overview

This folder contains all datasets, database files, data processing scripts, and documentation used throughout the project.

The objective of this phase is to collect, clean, validate, anonymize, and organize data for use in the SAP S/4HANA-inspired ERP implementation.

## Contents

### Raw Data

Original source files collected from operational and financial records before cleaning and transformation.

### Processed Data

Cleaned and standardized datasets prepared for analysis, reporting, forecasting, and dashboard development.

### Database

SQLite database files used to store and manage integrated ERP data.

### Data Dictionary

Documentation describing tables, fields, data types, and business definitions used within the project.

## Data Sources

The project utilizes a combination of:

* Sales and revenue data
* Accounts receivable data
* Financial reporting data
* Cost and budget data
* Procurement and inventory data
* Simulated ERP master data where required

## Data Security and Privacy

All business data used in this project is anonymized to protect organizational and customer confidentiality.

Sensitive information such as company names, customer identities, employee details, and financial figures are removed, modified, or masked before use.

* Data quality validation reports

* Data preparation scripts
