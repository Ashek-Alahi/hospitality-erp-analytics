# SAP FI Module

## Overview

This folder contains the Financial Accounting (FI) component of the SAP S/4HANA-inspired ERP implementation.

The FI module focuses on financial transaction processing, accounts receivable management, revenue tracking, cash flow monitoring, and financial reporting for hospitality operations.

## Scope

The implementation replicates selected SAP FI business processes using Python, SQLite, and Power BI.

Key areas include:

* Accounts Receivable (AR) Management
* Revenue Recognition and Tracking
* Financial Period Closing
* Cash Flow Analysis
* Financial Performance Reporting

## SAP References

The project is inspired by the following SAP FI functionalities:

* FI-AR (Accounts Receivable)
* FI-GL (General Ledger Accounting)
* Customer Account Analysis
* Financial Reporting and Reconciliation

## Key Features

### Accounts Receivable Management

* AR aging analysis
* Outstanding balance tracking

### Revenue Management

* Revenue by category

### Financial Reporting

* Period-end summaries
