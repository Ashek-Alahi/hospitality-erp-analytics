# Executive Summary

This portfolio MVP demonstrates how finance, controlling, sales, procurement, inventory, and forecasting data can be combined into a practical hospitality ERP analytics layer. The project is synthetic/anonymized and SAP S/4HANA-inspired; it is not a real SAP implementation.

## Key Findings

- Revenue is seasonal, with stronger performance in winter and summer hospitality periods.
- Accounts receivable monitoring highlights open balances that require collection follow-up.
- Cost center variance reporting identifies where operations are over or under budget.
- Procurement and inventory outputs connect vendor delivery performance to reorder risk.
- Forecasting baselines provide transparent, explainable planning metrics suitable for an MVP.

## Business Value

The repository gives recruiters and hiring managers a complete, runnable example of ERP analytics thinking: data model design, Python automation, SQL readiness, KPI reporting, and dashboard presentation without proprietary data or binary deliverables.
