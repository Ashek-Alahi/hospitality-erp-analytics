SAP S/4HANA-inspired hospitality ERP analytics prototype using synthetic data, Python, SQL, Markdown reporting, tests, SVG charts, and a static HTML dashboard.

## LinkedIn Featured description
Portfolio project demonstrating ERP/business analytics across FI, CO, SD, MM, forecasting, and executive KPI reporting for a fictional hospitality scenario using deterministic synthetic data.

## Portfolio website description
A text-first ERP analytics project that turns synthetic hotel operations data into finance, controlling, sales, procurement, inventory, forecasting, SQL, and dashboard outputs. Built for transparent GitHub review and interview discussion.

## Resume bullet points
- Built a SAP S/4HANA-inspired hospitality ERP analytics prototype using synthetic data, Python, SQL, Markdown reporting, and a static HTML dashboard.
- Designed FI, CO, SD, and MM analytics outputs covering AR aging, collections, cost-center variance, revenue segmentation, vendor performance, and inventory reorder risk.
- Automated deterministic data generation, validation, KPI reporting, forecasting baselines, and dashboard output with a one-command Python pipeline and pytest validation.

## Best interview positioning
Position the repository as evidence of ERP process understanding, accounting/business analytics judgment, technical execution, and professional communication. Emphasize that it is a portfolio prototype, not a live SAP implementation.

## What not to overclaim
Do not describe this as a real SAP project, production deployment, live customer deployment, direct SAP table extraction, advanced ML project, or Power BI project.
